# 学习分析仪表板 - 快速开始

## 🚀 立即体验

### 1. 启动应用
```bash
cd C:\Users\Administrator\online-game\CogniGuide\CogniGuide
npm run dev
```

### 2. 访问学习分析
1. 打开浏览器访问 http://localhost:5173
2. 在右侧"学习仪表板"标题旁，点击**图表图标**（📊）
3. 学习分析仪表板将打开

### 3. 探索功能

#### 📊 概览标签
- 查看整体学习统计
- 了解知识掌握分布
- 查看学习阶段分布

#### 📈 趋势分析标签
- 查看最近14天的学习趋势
- 识别最佳学习时段
- 了解知识点积累速度

#### 💡 智能洞察标签
- 发现学习强项和弱项
- 获取个性化学习建议
- 查看最佳学习时段

#### 🏆 成就标签
- 查看已解锁的成就
- 追踪未完成成就的进度
- 挑战自我解锁更多成就

### 4. 切换时间范围
使用顶部的时间范围按钮：
- **今天**: 查看今天的学习情况
- **本周**: 查看本周的学习趋势
- **本月**: 查看本月的学习统计
- **全部**: 查看所有历史数据

### 5. 导出学习报告
点击右上角"导出报告"按钮，下载详细的学习分析报告。

## 📊 数据可视化

### 主要图表
1. **知识掌握分布饼图**
   - 绿色 = 专家级
   - 蓝色 = 熟练级
   - 黄色 = 新手级
   - 灰色 = 未知

2. **学习趋势折线图**
   - 显示最近14天的学习时长
   - 帮助识别学习模式和习惯

3. **学习时段柱状图**
   - 24小时分布
   - 找出您最活跃的学习时间

4. **成就进度条**
   - 每个未解锁成就显示进度百分比
   - 激励持续学习

## 🎯 成就系统

### 解锁所有10个成就：
- 🎯 初学者
- 📚 勤奋学习者 (10次会话)
- 🏆 学习达人 (50次会话)
- 💡 知识收集者 (50个知识点)
- 🌟 知识大师 (100个知识点)
- 🎓 专家之路 (10个专家级知识点)
- 🔥 坚持不懈 (连续7天)
- ⏰ 专注一小时 (单次60分钟)
- ⏳ 学习积累 (累计5小时)
- 🎨 完整学习者 (5个阶段)

## 💡 使用技巧

1. **定期查看**: 每周查看一次学习分析，了解进度
2. **设置目标**: 基于洞察设置学习目标
3. **优化时间**: 根据时段分析优化学习安排
4. **关注弱点**: 重点关注"待提升领域"
5. **保持连续**: 每天学习一点点，解锁连续学习成就

## 🔧 技术细节

- **数据来源**: localStorage中的会话数据
- **图表库**: Recharts
- **样式**: Tailwind CSS
- **响应式**: 支持所有设备尺寸

## 📝 注意事项

- 首次使用可能数据较少，建议进行几次学习后再查看
- 学习时长基于消息时间戳估算，仅供参考
- 成就会自动解锁，无需手动操作
- 所有数据存储在本地，不会上传到服务器
