# 🚀 间隔重复学习系统 - 快速开始指南

## 30秒上手

### 第一次使用

1. **打开Dashboard右侧边栏**
2. **切换到"间隔重复"Tab**
3. **开始学习！**

---

## 📝 创建卡片的三种方式

### 方式1️⃣: 自动提取（推荐）

```
1. 与AI对话学习新知识
2. 点击"从笔记提取"按钮
3. 系统自动创建复习卡片
```

**优点**: 快速、准确、智能

### 方式2️⃣: 手动创建

```
1. 点击"手动创建卡片"
2. 填写问题和答案
3. 设置优先级和标签
4. 点击创建
```

**优点**: 灵活、个性化

### 方式3️⃣: 导入Anki格式

```
1. 点击"Anki 导入/导出"
2. 粘贴JSON数据或选择文件
3. 点击"导入卡片"
```

**优点**: 批量导入、数据迁移

## ⚡ 复习流程

### 日常复习（5分钟）

```
1. 看到红色提醒 → 点击"立即复习"
2. 点击卡片查看答案
3. 诚实选择评分：
   😫 完全忘记 (1分)
   😕 不记得 (2分)
   😐 困难回忆 (3分)
   😊 正确但犹豫 (4分)
   😎 轻松记住 (5分)
4. 自动进入下一张
```

### 评分指南

| 你的感受 | 选择评分 | 下次复习 |
|---------|---------|---------|
| 毫不费力想起 | 😎 5分 | 4天后 |
| 想起来了但有点犹豫 | 😊 4分 | 2-3天后 |
| 费了很大劲才想起 | 😐 3分 | 1天后 |
| 看到答案才想起 | 😕 2分 | 10分钟后 |
| 完全没印象 | 😫 1分 | 1分钟后 |

## 📊 查看学习进度

### 统计数据

在"间隔重复"Tab顶部可以看到：

- **总卡片数**: 你创建的所有卡片
- **待复习**: 需要现在复习的数量
- **今日已复习**: 今天完成的复习
- **平均质量**: 你的平均评分（目标：≥4.0）

### 记忆强度分布

```
[红 弱] [黄 中] [绿 强]
  3张    5张    12张
```

- **红色弱区**: 需要加强复习
- **黄色中区**: 稳步提升中
- **绿色强区**: 掌握牢固

## 💡 使用技巧

### 最佳实践

✅ **每日复习**: 设定固定时间（如每晚9点）
✅ **诚实评分**: 确保算法准确性
✅ **优先高优先级**: 红色标签的卡片优先复习
✅ **定期提取**: 每次学习后提取知识点

### 避免这些

❌ 不要连续给5分（会让间隔变得过长）
❌ 不要跳过低分卡片（这些是最需要复习的）
❌ 不要一次复习太多（建议每次≤20张）
❌ 不要只在到期时才复习（可以提前复习）

## 🎯 学习目标

### 第1周：建立习惯
- 每天创建5-10张卡片
- 每天复习1次
- 目标：平均质量≥3.5

### 第2-4周：巩固基础
- 每天复习15-20张卡片
- 重点复习低分卡片
- 目标：平均质量≥4.0

### 第2个月：长期记忆
- 每天复习20-30张卡片
- 绿色卡片占比>50%
- 目标：平均质量≥4.5

## 🔧 常见问题

### Q: 卡片太多复习不完怎么办？
A: 优先复习高优先级和到期卡片，可以设置每日上限

### Q: 总是忘记怎么办？
A: 说明难度因子偏低，多给3-4分，系统会自动调整

### Q: 如何删除不需要的卡片？
A: 目前可通过localStorage手动删除，后续会增加UI

### Q: 可以导出到Anki吗？
A: 可以！使用"Anki 导入/导出"功能

### Q: 复习数据会丢失吗？
A: 不会！所有数据保存在localStorage，即使关闭浏览器也不会丢失
