## 📊 数据处理

### 数据来源
从localStorage读取`cogniguide_sessions`：
- 每个会话的学习状态
- 知识点掌握度
- 消息历史
- 时间戳

### 学习时长计算
```
会话时长 = 最后消息时间 - 首条消息时间
总时长 = Σ(所有会话时长)
最少单位: 1分钟/会话
```

### 成就解锁
实时计算，自动解锁：
- 统计满足条件的会话/知识点
- 检查连续学习天数
- 计算累计时长
- 检查阶段覆盖
